## 2. Drawdown criterion behavior when cumulative gross P&L ≤ 0

Pre-committed criterion: "drawdown ≤ 30% of cumulative gross profit."
Denominator undefined when cumulative ≤ 0 (the present preview case:
dd $72.81 vs cumulative ~-$14).

Two natural rules:

- **a) auto-FAIL** the drawdown criterion when cumulative ≤ 0
  (interprets a strategy that hasn't broken even as having unbounded
  drawdown ratio)
- **b) auto-N/A** — drawdown criterion does not gate the verdict when
  cumulative ≤ 0; verdict rests on the other criteria alone
  (interprets drawdown as a profit-protection metric, only meaningful
  when there are profits to protect)

Different reasonable people would pick different defaults. Pick before
eval-execution.
